# Snake-Game
- It is is a classic Snake Game implementation using HTML, CSS, and JavaScript.
- It's a fun and interactive game that you can play directly in your web browser.
-  The game challenges you to control a growing snake as it navigates a grid, eating food and avoiding collisions with the walls.
## Game Instructions
- Use the arrow keys (Up, Down, Left, Right) on your keyboard to control the snake's direction.
- The snake will grow longer each time it eats a piece of food.
- Your goal is to eat as much food as possible without colliding with the walls or the snake's own body.
- The game ends when the snake collides with the walls.
